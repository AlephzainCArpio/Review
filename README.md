## Topics Covered

### ✅ Fundamentals
- Variables (`let`, `const`, `var`)
- Data types (strings, numbers, booleans)
- Conditionals (`if`, `else if`, `else`)
- Loops (`for`, `forEach`)
- Hoisting (var vs let/const, declarations vs expressions)
- Closures (lexical scope, factory functions, stateful functions)

### ✅ Functions
- Declarations vs expressions
- Arrow functions
- Parameters, arguments, return values
- Higher-order functions

### ✅ Arrays
- Indexing, `.length`, mutation
- `map`, `filter`, `reduce`
- Method chaining

### ✅ Objects
- Key-value pairs, dot notation
- Methods with `this`
- Arrays of objects
- Destructuring

### ✅ ES6+ Features
- Template literals
- Destructuring
- Spread and rest operators (`...`)
- Arrow functions

### ✅ Error Handling
- `try`, `catch`, `throw`
- Input validation
- Custom error messages

### ✅ Asynchronous JavaScript
- Promises (`resolve`, `reject`, `.then()`, `.catch()`)
- `async` / `await`

### ✅ DOM Manipulation
- Selecting elements (`getElementById`)
- `textContent`, `input.value`
- `addEventListener`
- Interactive UI (counter, input, button events)

### ✅ Version Control
- Git branch workflow
- Staging, committing, pushing
- Pull requests
